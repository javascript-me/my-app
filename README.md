
Install dependencies. Please ensure you have latest node [e.g. https://nodejs.org/dist/v6.10.1/] installed in your machine.

```
npm install
```

Run unit test.

```
npm test
```

Run application.

```
npm start
```

(Optional) Build to single optimized files and run.

```
npm run build
npm install -g serve
serve -s build
```
